# 问题描述

有一个进程 每隔一段时间频繁重启，由平台提供信息为 funcA 执行超时（平台有，函数执行时长检查，如果超过某一时长，则会退出，重启）。并给出了具体的具体的地址。



# 反编译
misp平台

# 和反编译出的地址比较， 通过平台给出的每次重启的位置发现，比较集中，怀疑是某一片代码引起的问题，函数执行超时，可能是死循环。
或者消耗资源的函数，执行时间太长。
通过合理推断排除 消耗资源的函数执行时间过长。
